## Project Overview

### What is SafeBasket?

SafeBasket v3 is an intelligent e-commerce product analysis platform that helps users make informed purchasing decisions by analyzing:

- **Product Reviews** (authenticity, sentiment, patterns)
- **Trust Scores** (reliability metrics)
- **Fake Review Detection** (AI-powered fraud detection)
- **Grievance Analysis** (complaint patterns)
- **Game Theory Decision Support** (strategic purchase recommendations)
- **Price Comparison** (multi-platform price discovery)

### Technology Stack

**Backend:**
- Flask (Python web framework)
- SQLAlchemy (Database ORM)
- SerpAPI (Review & price data extraction)
- JWT (Authentication)

**Frontend:**
- React (Single-page application)
- Babel (JSX transpilation)
- Tailwind CSS (Styling)

**AI/ML Components:**
- Sentiment Analysis (VADER)
- Fake Review Detection (Pattern recognition)
- Game Theory Engine (Nash Equilibrium analysis)
